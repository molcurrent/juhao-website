# 旧站安全与 SEO 上线门禁

核验日期：2026-07-12

## 已确认风险

搜索结果仍可发现 `www.juhao.com/static/news/*.html` 下的博彩关键词与第三方医院内容。已确认样本记录在 `content/governance/legacy-url-actions.csv`。这些地址不能 301 到新首页，应在旧站清理内容后返回 410，并向搜索平台提交移除。

## 必须由旧站管理员执行

1. 在隔离环境备份旧站文件、数据库、访问日志与 Web 服务器配置。
2. 核查后台账号、上传目录、定时任务、异常 PHP 文件、模板注入和数据库异常文章。
3. 清除后门与垃圾内容，轮换服务器、数据库、后台、CDN、对象存储和发布凭证。
4. 将确认的垃圾 URL 配置为 410；合法旧页面逐条建立 301 映射；商城登录、订单和支付地址保持可用。
5. 在百度、必应等搜索平台提交垃圾 URL 移除和新 sitemap，并持续观察新增异常目录、404、索引量与品牌关键词。

## 上线判定

- 旧站管理员提供恶意文件清单、处置记录和凭证轮换记录。
- `legacy-url-actions.csv` 中全部 spam URL 在线返回 410。
- 合法 URL 映射经抽样验证，不存在全部跳转首页的软 404。
- 新站 sitemap、canonical、结构化数据和统计配置已验证。

当前仓库只完成风险样本台账与新站 SEO 结构，未获得旧服务器和搜索平台权限，因此不能声称旧站已经清理完成。
